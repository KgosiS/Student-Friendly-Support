# Student Friendly Support

A modern, student-focused landing page for Student Friendly Support, a service that helps students with programming assignments and university applications.

## Overview

This project is a responsive one-page website built with HTML, CSS, and JavaScript. It highlights:

- Programming assignment support for Python, Java, C#, Android, and web development
- University application assistance
- A contact form that opens a WhatsApp message
- A clean, light-themed design with smooth scrolling

## Project Structure

- `index.html` — main page content
- `css/style.css` — site styling and responsive layout
- `js/script.js` — smooth scrolling, mobile navigation, and WhatsApp form handling
- `assets/` — folders for icons, images, and vector assets

## Features

- Responsive navigation
- Smooth scrolling between sections
- Light-themed UI
- Service cards with technology icons
- Contact form that opens WhatsApp

## How to Run Locally

1. Open the project folder in your browser, or use a simple local server.
2. If you prefer a local server, run:

```bash
python -m http.server 8000
```

3. Then open:

```text
http://localhost:8000
```

## Customization

You can update:

- Page content in `index.html`
- Colors and layout in `css/style.css`
- Contact number and form behavior in `js/script.js`

## Notes

The contact form currently uses a sample WhatsApp number. Replace it with your real number in `js/script.js` if needed.
